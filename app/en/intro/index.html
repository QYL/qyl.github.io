Introducing Dialogue App

What is Dialogue?

Dialogue is a subtitle reader. Its goal is to help users who like to learn English through watching films or TV series to improve their learning efficiency.

Features

Dialogue is a minimalist application that doesn’t have a lot of features, but the features it has are very useful.

Subtitle File Parsing

The most important feature of Dialogue is subtitle file parsing. She can automatically parse complex .srt and.ass files and render them into clean and readable texts. While reading, if you encounter a new word, you can press and hold to query its definition. This design eliminates the need for users to constantly jump between different applications during reading and improves the reading experience.

Merging Chinese and English Lines

Some subtitle files contain Chinese and English dialogues, but their contents are not on the same line. For example, the first few hundred lines are Chinese and the last few hundred lines are English. This kind of file is very inconvenient to read. The Chinese / English Merging function of Dialogue can automatically merge those lines during the parsing process, so that Chinese and English lines appear at the same section.

It is worth mentioning that this feature is currently in an experimental stage, so this feature is turned off by default, and turning on this feature may slow down the parsing process. However, Dialogue internally caches the parsing results. The same file only needs to be parsed once, so the impact on normal use should not be a problem. To use this feature, users need to manually turn it on in the settings page.

Bookmark New Words

You can also collect new words, and if you enabled the context selector in the settings you may reserve their context. By enabling the context selector, a context selector will pop up when bookmarking new words. You can use it to pick the context in which the word is located. This can ensure that the example sentence for the word your are collecting is a complete sentence, rather than a part of the sentence.

Content Export

Dialogue supports exporting subtitle content to PDF files, while new words can be exported to PDF or CSV files. PDF or CSV files can be managed (downloaded, deleted) directly in the web interface for uploading subtitles, and files can also be copied or shared to other apps via the share sheet.

Wireless Upload

Dialogue Supports uploading subtitle files to the app via Wi-Fi. It is an easy task to manage the files.

Sync with iCloud

Dialogue supports iCloud syncing. You can drop your subtitle files in the Dialogue directory under iCloud Drive on your Mac, and those files will be automatically synced to your iPhone/iPad.

Import from Other Apps

Download subtitle files in other applications and call out the share sheet within the app. Then select Dialogue in the pop-up menu to import the subtitle file into Dialogue.

Fast Scrolling

A round button appears as you scroll through the page. Drag it to scroll quickly.
