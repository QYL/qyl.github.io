使用教程

簡介

歡迎使用對白。本文將簡單介紹對白的使用方法。我們首先需要明確一點，對白是一款閱讀器，本身並不自帶內容。因此，我們必須把我們想要閱讀的字幕文件(.ass 或 .srt)上傳到對白應用內。

文件上傳

在上傳文件之前，你的設備必須滿足两個條件：(1) 你的手機和電腦連接了同一個Wi-Fi；(2)手機沒有設置代理。

在滿足以上條件之後，便可以上傳字幕文件了。此時點擊 App 主界面右上角的 “+” 按鈕，應用會彈出一個提示框，上面有上傳頁面的地址。

在瀏覽器地址欄輸入對應的地址（包括端口號），即可進入文件上傳頁面。在上傳頁面完成上傳之後，關閉上傳提示框，便可看到剛剛上傳的文件出現在了文件列表中。

字幕解析

如果你直接用文本編輯軟件打開.ass或者.srt字幕文件，就會發現裡面除了字幕內容和時間軸，還有一大堆的編碼，這些其實都是一些特效和样式。這些內容在視頻中顯示時可以讓字幕更加美觀，但是對於我們閱讀的人而言則是噪音。 對白會像變魔術般地將這些繁雜的內容全部過濾掉，只留下字幕的內容，並以中英對照(前提是你上傳是的雙語字幕)的方式呈現給用戶。用戶需要做的只有一步，即在文件列表中點擊自己想要閱讀的字幕文件，應用便會自動開始解析。需要注意的是，第一次解析可能需要耐心等待幾秒的時間，之後打開則幾乎無需等待。

單詞查詢

在閱讀過程中，如果遇到不認識的單詞長按即可查詢定義。這一設計免去了用戶閱讀期間不斷在不同應用之間跳轉的煩惱，提升了閱讀體驗。另外，你還可以收藏生詞，並且生詞自帶上下文。

自定義生詞上下文

在設置中開啟上下文選擇器之後，在添加生詞時會彈出一個上下文選擇器，你可以通過它選擇單詞所在的上下文，這樣可以保證單詞例句是一個完整的句子，而不是句子的一部分。

內容導出

對白支持將字幕內容導出到 PDF 文件，而生詞本則可以導出到 PDF 或者 CSV 文件。 PDF 或者 CSV 文件可以直接在上傳字幕的 Web 界面進行管理（下載、刪除），也可以通過分享菜單拷貝、分享到其他 app 中使用。如需導出內容，只需在生詞本或者字幕閱讀頁麵點擊右上角的分享按鈕即可生成文件。你可以開啟 Wi-Fi 傳字幕的功能，然後在 Web 界面管理生成的文件，或者使用文件生成之後彈出的分享功能菜單，將文件分享、拷貝至其他應用。

文件同步

對白支持 iCloud 同步字幕文件。現在只需在 Mac 上把字幕文件往 iCloud Drive 下的 Dialogue 目錄一扔，文件就會自動同步到 iPhone 上。

外部導入

在其它應用內點擊字幕文件，選擇在其他應用中打開，然後在彈出菜單中選擇對白即可將字幕文件導入到對白。

快速滾動

閱讀頁面滾動時會出現一個圓形按鈕，拖動即可快速滾動。

其他

對白作為一款閱讀工具，當然不能少了字典和夜間模式。在閱讀時，長按即可選擇需要查詢的單詞。在燈光較暗的環境中閱讀時，可以到 App 設置頁面打開夜間模式，以保護你的眼睛。
